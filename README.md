# 💾 Retro Web — Windows 95/98 Style

Proyecto web experimental que recrea la estética del escritorio de **Windows 95 / Windows 98** utilizando tecnologías web básicas.

La idea es simular una interfaz retro con ventanas, escritorio, barra de tareas y elementos clásicos del sistema operativo dentro del navegador.

---

## 🖥 Demo

Este proyecto se ejecuta localmente en el navegador.

Ejecutar `index.html` o utilizar **Live Server** en VS Code.

---

## ⚙ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Visual Studio Code
- Live Server

## 📚 Objetivo del proyecto

Este proyecto fue creado con fines de aprendizaje y experimentación para:

- Practicar **HTML, CSS y JavaScript**
- Comprender manipulación del DOM
- Simular interfaces gráficas dentro del navegador
- Recrear interfaces retro utilizando CSS
